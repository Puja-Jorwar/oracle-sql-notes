\# Oracle SQL Projection Assignment



\## 01) WAQTD ALL THE DETAILS FROM THE EMPLOYEE TABLE

```sql

SELECT *

FROM EMP;

```



\## 02) WAQTD NAMES OF ALL THE EMPLOYEES

```sql

SELECT ENAME

FROM EMP;

```



\## 03) WAQTD NAME AND SALARY GIVEN TO ALL THE EMPLOYEES

```sql

SELECT ENAME, SAL

FROM EMP;

```



\## 04) WAQTD NAME AND COMMISSION GIVEN TO ALL THE EMPLOYEES

```sql

SELECT ENAME, COMM

FROM EMP;

```



\## 05) WAQTD EMPLOYEE ID AND DEPARTMENT NUMBER OF ALL THE EMPLOYEES IN EMP TABLE

```sql

SELECT EMPNO, DEPTNO

FROM EMP;

```



\## 06) WAQTD ENAME AND HIREDATE OF ALL THE EMPLOYEES

```sql

SELECT ENAME, HIREDATE

FROM EMP;

```



\## 07) WAQTD NAME AND DESIGNATION OF ALL THE EMPLOYEES

```sql

SELECT ENAME, JOB

FROM EMP;

```



\## 08) WAQTD ENAME, JOB, SALARY GIVEN TO ALL THE EMPLOYEES

```sql

SELECT ENAME, JOB, SAL

FROM EMP;

```



\## 09) WAQTD DNAME AND LOCATION PRESENT IN DEPT TABLE

```sql

SELECT DNAME, LOC

FROM DEPT;

```



\## 10) WAQTD DNAME PRESENT IN THE DEPT TABLE

```sql

SELECT DNAME

FROM DEPT;
```

\# Oracle SQL Expression and Alias Assignment



\## 01) WAQTD NAME OF THE EMPLOYEE ALONG WITH THERE ANNUAL SALARY

```sql

SELECT ENAME, SAL\*12 AS ANNUAL\_SALARY

FROM EMP;

```



```

ENAME   ANNUAL\_SALARY

SMITH   9600

ALLEN   19200

WARD    15000

...

```



\## 02) WAQTD ENAME AND JOB FOR ALL THE EMPLOYEE WITH THERE HALF TERM SALARY

```sql

SELECT ENAME, JOB, SAL\*6 AS HALF\_TERM\_SALARY
```

```
ENAME  JOB       HALF\_TERM\_SALARY
```
